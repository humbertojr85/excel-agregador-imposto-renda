# excel-agregador-imposto-renda
Ferramenta desenvolvida em Excel para organizar e centralizar informações essenciais para a declaração do Imposto de Renda. O projeto utiliza validações automáticas, menus de navegação e recursos de usabilidade para garantir um preenchimento eficiente, seguro e intuitivo, aplicando na prática os conceitos aprendidos durante o desafio da DIO.
